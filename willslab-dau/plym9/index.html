Data and Analysis Unit: PLYM9

Last update: 2014-12-19 by Andy Wills

Description

Analysis of model consistency and margin for Experiments 1-5 of Wills, Inkster, and Milton (submitted). This DAU contains analysis scripts in open cross-platform formats (see the file format notes).

Citation

The APA-format citation for this resource is: Wills, A.J. (2014). Data and Analysis Unit: PLYM9. Retrieved from http://www.willslab.co.uk/plym9

If you make use of these resources, please drop me an email: andy@willslab.co.uk.

Resources

  • plym9analysis.R (1 KB) - R script for the analysis of consistency and margin, averaged across all reported triad experiments. Requires .RData files generated by R scripts for each experiment (see other DAUs).
  • plym9analysis2.R (1 KB) - R script for the analysis of consistency and margin, averaged across all reported criterial-attribute experiments. Requires .RData files generated by R scripts for each experiment (see other DAUs).
