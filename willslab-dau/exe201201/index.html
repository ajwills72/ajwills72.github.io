Data and Analysis Unit: EXE201201

Description

Category learning experiment with observational and feedback training, reported by Edmunds, Milton & Wills (submitted), Experiment 1.

Citation

The APA-format citation for this resource is: Edmunds, C. E. R., Milton, F. N., & Wills, A. J. (2014). Data and Analysis Unit: EXE201201. Retrieved from http://www.willslab.co.uk/exe201201

Resources

EXE201201.csv (1.8 MB) - Trial-level raw data. Column headings are as follows:
  • Participant: Participant Number.
  • Condition: 1=Conjunction-Observational training, 2=Conjunction-Feedback training, 3=Information-integration-Observational training, 4=Information-integration-Feedback training.
  • Block: Block number from 1 to 10.
  • Phase: Experiment phase (1=Training, 2=Test).
  • Trial: Trial number (does not reset for each block).
  • Stimulus: Arbitrary, stimulus-unique, ID code.
  • Length: Stimulus length measured in pixels.
  • Orientation: Stimulus orientation in arbitrary coordinates. They were converted into angles measured in radians, anti-clockwise from East, by applying a scaling factor of pi/500.
  • Category: Correct category assignment for each stimulus (1=Category A, 2=Category B).
  • Response: The response on each trial (29=Category A, 56=Category B).
  • Correct: Whether the response was correct or not (1=Correct, 0=Incorrect).
EXE201201Verbal.csv (3 KB) - Verbal report data for each participant. There is data missing from one participant. Column headings are as follows:
  • Participant: Participant Number.
  • Established: Whether they established a strategy or not (Y/N).
  • Strategy: UDL-unidimensional length, UDO-unidmiensional orientation, CJ-upper-left conjunction, CJ2-specified both upper-left and lower-right conjunctions, 2D-mentioned both stimulus dimensions, O-other.
  • Condition: CJObserve=Conjunction-Observational training, CJFeedback=Conjunction-Feedback training, IIObserve=Information-integration-Observational training, IIFeedback=Information-integration-Feedback training.
EXE201201Verbal.pdf (1.3 MB) - Verbal report data for each participant. Participant number at top left.
