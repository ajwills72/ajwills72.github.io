Data and Analysis Unit: PLYM4

Last update: 2014-06-06 by Andy Wills

Description

Analysis of impulsivity data reported in Experiments 2, 3A and 3B of Wills, Inkster, and Milton (submitted). The analyses are presented in the Supplementary Materials. This DAU contains summary files and an analysis script, both in open cross-platform formats (see the file format notes).

Citation

The APA-format citation for this resource is: Wills, A.J. (2014). Data and Analysis Unit: PLYM4. Retrieved from http://www.willslab.co.uk/plym4

If you make use of these resources, please drop me an email: andy@willslab.co.uk.

Resources

  • plym4analysis.R (4 KB) - R script for analyses reported in Wills, Inkster, Milton (submitted). This script can be run by the command source('plym4analysis.R'), or can be run interactively.
  • plymsum.tbz (8 KB) - Summary data files in .RData format. These are generated by the analysis scripts for PLYM1,2, and 3, and are provided here for convenience as plym4analysis.R requires them to run. They are intermediate files, NOT raw data.
