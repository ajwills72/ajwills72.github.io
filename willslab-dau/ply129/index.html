Data and Analysis Unit: PLY129

These resources have not yet been published. If you access them, you are agreeing to treat the information you receive in confidence (e.g. as part of a peer review process). Once the manuscript is accepted, these same resources will be made publicly available.

Description

Test-potentiated learning experiment examining recall performance, reported in Seabrooke, Hollins, Wills, Mitchell (submitted), Experiment 2. This DAU contains raw data and analysis scripts, in open cross-platform formats (see the file format notes). The raw data file also contains the full set of stimulus items, as the stimuli presented on each trial are recorded (see below).

Resources

  • TPL2_LongData.csv (445 KB) - Trial-level raw data.Column headings are as follows:
    • Subject - Participant number, unique to ply129
    • Experiment - Experiment name
    • Group - Counterbalancing condition for order of encoding phases (e.g., Study, First Word, Meaning)
      1. S,M,F
      2. S,F,M
      3. M,S,F
      4. M,F,S
      5. F,S,M
      6. F,M,S
    • Age - Age of participant (in years)
    • Gender - Gender of pariticipant (female, male)
    • Running - Combines phase and w/subj condition of experiment. [Study, Meaning, First Word] indicate an encoding trial in the corresponding condition. [Recall] indicates a cued recall trial.
    • Condition - In the recall phase, the encoding condition under which the word was presented. In the encoding phase, set to NA.
    • Cycle - Recall test number. Always set to one in the encoding phase.
    • Sample - Within-phase trial number (resets at the beginning of each new encoding phase, continues from previous cued recall phase).
    • Cue - In the encoding phase, the presented cue word. In the recall phase, the presented retrieval cue.
    • Target - The presented target word.
    • Acc - Whether the participant's response is correct (1) or incorrect (0). In the Study condition of the encoding phase, set to NA. In cued recall, the word must match letter-for-letter in order to be coded as 1.
    • Resp - The participant's response.
    • RT - The time (in ms) from onset of the cue to the participant pressing the last key. In the Study condition of encoding phase, set to NA.
  • R scripts for analysis of PL129. You will need R. You will also need to have installed the following packages that are available on CRAN: ez, ggplot2, pastecs, plyr.
    • TPL2Functions.R - functions required by the other script, below.
    • tpl2-analysis.R - All statistical analysis reported in the mansucript.
