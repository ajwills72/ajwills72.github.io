Data and Analysis Unit: PLYM1

Last update: 2014-11-21 by Andy Wills

Description

Triad classification task, reported in Experiment 2 of Wills, Inkster, and Milton (submitted). This DAU contains stimulus files, raw data, and an analysis script, all in open cross-platform formats (see the file format notes).

Citation

The APA-format citation for this resource is: Wills, A.J. (2014). Data and Analysis Unit: PLYM1. Retrieved from http://www.willslab.co.uk/plym1

If you make use of these resources, please drop me an email: andy@willslab.co.uk.

Resources

  • plym1stim.tbz (20 KB) - Stimulus image files.
  • plym1qu.txt (1 KB) - Statements for Impulsivity questionnaire
  • plym1data.txt (225 KB) - Trial-level raw data. Column headings are as follows:
    • cond: Experimental condition (1 = high time pressure, 2 = low time pressure)
    • subj: Participant ID (unique within PLYM1)
    • phase: Experiment phase (1 = triad classification, 2 = impulsivity questionnaire)
    • blk: Block (always = 1 in phase 2. Ranges from 1 to 12 in phase 1)
    • trial: Trial number (resets for each block)
    • stim: Phase-specific stimulus code:
      • Phase 1 = ID of presented stimulus triad (see plym1code.txt)
      • Phase 2 = zero (Impulsivity questions set in the fixed order shown in plym1qu.txt)
    • resp: Phase-specific response code
      • Phase 1: 1 = left , 2 = middle, 3 = right
      • Phase 2: 1 = rarely/never, 2 = occasionally, 3 = often, 4 = almost always
    • rt: Reaction time in milliseconds
  • plym1analysis.R (3 KB) R script for analyses reported in Wills, Inkster, Milton (submitted). This script can be run by the command source('plym1analysis.R'), or can be run interactively.
  • plym1code.txt (1 KB) - The main purpose of this file is to facilitate automated analysis of plym1data.txt by plym1analysis.R. Column headings are as follows:
    • triad: Triad ID (as used in plym1data.txt)
    • triplet: Triplet ID. Any given stimulus triplet can be presented in six different ways (three stimuli can be placed in three physical locations in six different ways). Hence, there are six different stimulus triad IDs for each triplet ID.
    • left1: Stimulus ID for stimulus presented at the left of the triad. Stimulus ID is a two-digit co-ordinate code in arbitrary units. For example Stimulus ID 25 indicates a value of 2 on the brightness dimension and a value of 5 on the size dimension. Larger values on the brightness dimension are associated with increasing brightness. Larger values on the size dimension are associated with increasing size. The mapping between these stimulus IDs and the numbering of the graphic files in this archive, are as follows: 1 = 11, 2 = 51, 3 = 22, 4 = 42, 5 = 24, 6 = 44, 7 = 15, 8 = 55
    • mid2: Stimulus ID for stimulus presented in the middle of the triad.
    • right3: Stimulus ID for stimulus presented at the right of the triad.
    • bright: Response expected if classifying on the basis of brightness ( 1 = left , 2 = middle, 3 = right )
    • size: Response expected if classifying on the basis of size
    • os: Response expected if classifying on the basis of overall similarity
    • ident: Response expected if classifying on the basis of the identical attribute.
  • barrattkey.txt (1 KB) - This file facilitates automated analysis of the impulsivity rating data. Column headings are as follows:
    • qu - Question number in the Barrat Impulsivity Questionnaire, see plym1qu.txt
    • att - Attentional subscale (0 = This qu. is not on this subscale, 1 = normal coded qu on this subscale, -1 = reverse coded qu on this subscale)
    • mot - Motor subscale
    • non - Nonplanning subscale
