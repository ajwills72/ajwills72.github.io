Data and Analysis Unit: PLY27

Description

Category learning experiment examining learning of a conjunction category structure with observational and feedback training, reported by Edmunds, Milton & Wills (submitted), Experiment 2.

Citation

The APA-format citation for this resource is: Edmunds, C. E. R., Milton, F. N., & Wills, A.J. (2014). Data and Analysis Unit: PLY27. Retrieved from http://www.willslab.co.uk/ply27

Resources

PLY27.csv (1 MB) - Trial-level raw data. Column headings are as follows:
  • Participant: Participant Number.
  • Condition: 1=Conjunction-Feedback training, 2=Conjunction-Observational training.
  • Block: Block number from 1 to 10.
  • Phase: Experiment phase (1=Training, 2=Test).
  • Trial: Trial number (does not reset for each block).
  • Stimulus: Arbitrary, stimulus-unique, ID code.
  • Length: Stimulus length measured in pixels.
  • Orientation: Stimulus orientation in arbitrary coordinates. They were converted into angles measured in radians, anti-clockwise from East, by applying a scaling factor of pi/500.
  • Category: Correct category assignment for each stimulus (1=Category A, 2=Category B).
  • Response: The response on each trial (68=Category A, 75=Category B).
  • Correct: Whether the response was correct or not (1=Correct, 0=Incorrect).
PLY27Verbal.csv (<1 KB) - Verbal report data for each participant. Column headings are as follows:
  • Participant: Participant Number.
  • Condition: 1-Feedback training, 2-Observation training
  • Strategy: UDL-unidimensional length, UDO-unidmiensional orientation, CJ-upper-left conjunction, CJ2-specified both upper-left and lower-right conjunctions, 2D-mentioned both stimulus dimensions, O-other.
  • Confidence: Answer to the question 'how sure are you that your strategy is the best one'. Likert scale from 1 (not at all) to 5 (Certain).
PLY27Verbal.pdf (1.7 MB) - Verbal report data for each participant. Participant code in top left hand corner. Those beginning with 2 were in the feedback training condition, 3 in the observational training condition.
