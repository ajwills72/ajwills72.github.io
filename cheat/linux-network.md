## Performance monitoring

| command      | availability | Explanation                                 |
| ------------ | ------------ | ------------------------------------------- |
| `ping <URL>` | base system  | Measure connection latency and reliability  |
| `speedtest`   | apt install speedtest-cli  | Get connection's upload and download speed  |
